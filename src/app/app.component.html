NG Firebase View

Purpose of the Project:

Visualise your Firebase Database locally.

Why?

  • If you have a local firebase (for example firebase-server) and no other way to inspect it

Features

  • Displays your Firebase data
  • Breadcrumb nav - select deeper nodes as startpoint
  • search for text
  • add shortcuts
  • changelog - highlight time relevant events
  • changelog - select change -> jump to dataTree

Installation - inside your application

  1. install via npm
  2. import module
  3. place wrapper component (optional)
  4. initialize via Service
